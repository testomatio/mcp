# Testomat.io MCP Enterprise

Enterprise Model Context Protocol server for Testomat.io API v2.

This package includes all tools from `@testomatio/mcp` plus enterprise analytics tools:

- `analytics_tests`
- `analytics_stats`

## Installation

```bash
npm install -g @testomatio/mcp-enterprise@latest
```

## Usage

```bash
testomatio-mcp-enterprise --token <PROJECT_TOKEN> --project <PROJECT_ID>
```

Analytics endpoints require the `api_analytics` subscription feature.
